# Appointment Management System

Full stack web application for managing massage therapy appointments. Users can create an account, book appointments, view their reservations and cancel them.

---

## 🚀 Features

- User registration and authentication
- Appointment booking with date and time selection
- View personal appointments
- Cancel appointments
- Data persistence in database
- Full stack application (Frontend + Backend)

---

## 🛠 Technologies

### Backend
- Node.js
- Express.js
- TypeScript
- PostgreSQL

### Frontend
- React
- HTML
- CSS

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd <project-folder>

🔧 Backend Setup

cd back
npm install

Create a .env file inside the back folder based on .env.example.

PORT=3000
DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=your_user
DB_PASSWORD=your_password
DB_DATABASE=your_database
DB_logging=
DB_synchronize= 
DB_dropSchema=

Run backend:
npm start

💻 Frontend Setup
cd front/vite-project
npm install
npm run dev
